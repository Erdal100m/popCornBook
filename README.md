Ключевые особенности этой верстки:Фоновый градиент: 
Написан в блоке body, цвет меняется от оранжевого в самом низу к белому наверху через нежно-оранжевый.
Фон слайдов: В качестве фона карточек книг выступает изображение попкорна, поверх которого добавлен полупрозрачный белый цвет, чтобы текст читался легко.Адаптивный слайдер: На больших экранах отображаются 3 книги одновременно, на планшетах — 2, на смартфонах — 1 (всего 6 слайдов в цикле).Навигация: В шапке есть кнопки "Книги" и "Контакты", а для мобильных устройств добавлена базовая иконка гамбургер-меню. 
По умолчанию она открывает простое окно-уведомление (при желании можно подключить логику выпадающего меню).
